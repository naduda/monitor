# GulpBowerTemplate

Basic Usage
1. npm i
2. bower i
3. gulp (Run default task for developing)
4. gulp build

Protractor
1. npm install protractor -g
		protractor --version
2. npm -g list
		selenium-webdriver
3. npm install selenium-webdriver -g
		selenium-webdriver
4. webdriver-manager update

RUN: webdriver-manager start
